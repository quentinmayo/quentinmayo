---

## Thoughts of an Engineer

The world of coding has changed.

Technical skill still matters, but now compute, tokens, tooling, and iteration speed matter too. AI has become a real force multiplier for building, testing, prototyping, and exploring ideas faster than before.

For a long time, much of my work lived behind company GitHub organizations. That work mattered, but it also meant my personal GitHub did not really reflect how much I was building, learning, and experimenting. That honestly sucked.

This account is part of changing that.

I’m using this space to build more openly, experiment more publicly, and turn ideas into code — even when they are rough, incomplete, or still forming.

I do care about security and engineering quality.

That said, many of these projects move fast. Some are experiments. Some are prototypes. Some are things I built to test an idea or explore a workflow.

Not everything here is production-ready, fully audited, hardened, or maintained.

Use anything in these repositories at your own risk.
